#Human-tec 웹사이트 업그레이드 프로젝트

##개요
본 프로젝트는 기존 imweb 기반의 구형 Human-tec 웹사이트를 Vercel의 AstroWind 템플릿을 사용하여 현대적이고 빠르며 확장 가능한 플랫폼으로 업그레이드하는 것입니다.

-**기존 사이트 참고**: https://humantec.imweb.me/

-**템플릿 참고**: https://vercel.com/templates/portfolio/astrowind
**(매우 중요) 위의 AstroWind 템플릿은 이미 본 프로젝트 폴더에 이식 완료 된 상태**

##목표 1.**UI/UX 현대화**: AstroWind의 깔끔하고 반응형인 디자인을 활용하여 기존 사이트의 투박한 요소를 제거합니다.2.2.2.**브랜드 일관성**: Human-tec의 기업 로고 색상인 **옐로우(Yellow), 그린(Green), 네이비(Navy)**를 기준으로 홈페이지 전체 컬러과 가장 조화로울 수 있도록 템플릿 컬러를 재디자인합니다. 3.**미래 지향적 설계**: 다국어 지원(한국어, 영어, 일본어) 및 사용자 인증 시스템(로그인/관리자 대시보드) 도입을 위한 아키텍처를 준비합니다.

##주요 커스텀 사항공지

- [ ] **바 제거**: 모든 페이지 상단의 검은색 공지 스트립을 삭제합니다.
- [ ] **헤더 수정 1 (다국어)**: Wi-Fi/RSS 아이콘을 언어 선택 드롭다운(한국어, 영어, 일본어)으로 교체합니다.
- [ ] **헤더 수정 2 (인증)**: 기존 'Download' 버튼을 'Login' 버튼으로 교체하여 향후 사용자 권한 및 관리자 기능을 준비합니다.
- [ ] **테마 통합**: Tailwind 설정을 통해 옐로우, 그린, 네이비 색상 체계를 전역에 적용합니다.
- [ ] **내비게이션 매핑**: AstroWind의 기본 메뉴를 Human-tec의 핵심 사업 카테고리에 맞춰 재구성합니다.
- [ ] **콘텐츠 마이그레이션**: 기존 사이트의 텍스트와 이미지를 가져오되, Astro의 이미지 컴포넌트와 Markdown/MDX 콘텐츠 컬렉션에 최적화하여 이식합니다.

## 시작 가이드

작업 시작 시, 먼저 현재 tailwind.config를 분석하고 색상 체계를 업데이트하십시오. 그 후 "주요 커스텀 사항" 목록을 체계적으로 수행하되, Header.astro 컴포넌트를 우선적으로 작업하십시오. 각 주요 항목을 완료할 때마다 사용자에게 검토를 요청하십시오.
